WebPage vs Website

เรียนรู้ความแตกต่างระหว่าง WebPage (หน้าเว็บเดียว) และ Website (หลายหน้า) ผ่านการทดลองใช้งานจริง

WebPage

หน้าเว็บเดียว (Single Web Page)

WebPage คือ เอกสารเดียวที่แสดงผลบนเว็บเบราว์เซอร์ มีเนื้อหาคงที่ และมักจะมีข้อมูลเฉพาะเรื่องใดเรื่องหนึ่ง เช่น บทความ ข่าว หรือข้อมูลผลิตภัณฑ์

บทความเรื่อง "การเรียน HTML"

นี่คือตัวอย่างของ WebPage ที่มีเนื้อหาเดียว คุณสามารถคลิกได้ แต่จะไม่มีการเปลี่ยนแปลงหน้า เพราะเป็นหน้าเดียวเท่านั้น

คลิกแล้ว! แต่ยังอยู่หน้าเดียว

คุณสมบัติของ WebPage:

  • มีเนื้อหาเดียว
  • ไม่มีการนำทางระหว่างหน้า
  • โหลดเร็ว
  • เหมาะสำหรับข้อมูลเฉพาะเรื่อง
  • URL เดียว
Website

เว็บไซต์ (Complete Website)

Website คือ ชุดของหลาย ๆ WebPage ที่เชื่อมโยงกันภายใต้โดเมนเดียวกัน มีระบบนำทาง และสามารถแสดงเนื้อหาที่หลากหลายได้

คุณสมบัติของ Website:

  • มีหลายหน้า (Multiple Pages)
  • มีระบบนำทาง (Navigation)
  • เนื้อหาหลากหลาย
  • มีโครงสร้างซับซ้อน
  • หลาย URL ภายใต้โดเมนเดียว
